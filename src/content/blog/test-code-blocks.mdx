## Output blocks

Simple output from a JavaScript function:

```js title="fibonacci.js"
function fib(n) {
  if (n <= 1) return n;
  return fib(n - 1) + fib(n - 2);
}

console.log([0, 1, 2, 3, 4, 5, 6, 7].map(fib));
// [output]
[0, 1, 1, 2, 3, 5, 8, 13];
```

Python with multi-line output:

```python title="hello.py"
name = "world"
print(f"Hello, {name}!")
for i in range(3):
    print(f"  count: {i}")
# [output]
Hello, world!
  count: 0
  count: 1
  count: 2
```
